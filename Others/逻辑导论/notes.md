# 逻辑导论
## 逻辑学是研究推理和论证的科学
### 什么是逻辑学
逻辑的意思
- 客观事物的规律
- 某种理论、观点
- 思维的规律、规则
- 逻辑学或逻辑知识

逻辑学的对象：推理与论证  
涉及思想之间的过渡、变换和迁移  
明白什么样的推理和论证是正确的、有效的

### 推理、推理形式及其有效性
1. 推理  
推理是从一个或一些已知的命题得出新命题的思维过程或思维形式  
前提 -> 结论
- 演绎推理  
根据某种一般性原理和个别性例证，得出关于该个别性例证的新结论  
例证真则结论真
- 归纳推理  
从一定数量的个别的真实，抽象概括出某种一般性原理  
是或然性推理，前提真结论不一定真
2. 推理形式  
指在一个推理中抽掉各个命题的具体内容之后所保留下来的模式或框架  
由逻辑常项和变项两部分构成

分析推理的三种方式
1. 停留在简单语句平面，不深入语句内层  
简单语句用p, q, r, s等表示  
简单语句用逻辑联结词联结起来构成复合语句  
$\neg$, $\vee$, $\wedge$, $\rightarrow$, $\leftrightarrow$
2. 深入语句内层，但把简单句分析成主谓式语句  
所有的S都是P: SAP  
所有的S都不是P: SEP  
有些S是P: SIP  
有些S不是P: SOP  
a是P: $a\in P$  
a不是P: $a\notin P$
3. 深入语句内层，但做与主谓式不同的另一种分析  
个体常项: a, b, c, ...  
个体变项: x, y, z, ...  
谓词符号: F, G, R, S, ...  
量词: $\forall$, $\exists$  
联结词: $\neg$, $\vee$, $\wedge$, $\rightarrow$, $\leftrightarrow$
